## Overview

This repository contains a pixel-perfect clone of a modern portfolio website. The challenge was to reproduce every layout detail, typography choice, spacing, and visual hierarchy from the original design while preserving clean, maintainable code.

The final product demonstrates my ability to build responsive, visually compelling websites with a focus on precision and attention to detail. It is a tablet and laptop first responsive design, showcasing front-end craftsmanship from navigation flow and hero section presentation to project previews and skill highlighting.

## What This Clone Shows

- Pixel-perfect replication of the original portfolio layout
- Responsive structure that adapts across tablets, and laptop
- Strong visual hierarchy and clear section separation
- Careful attention to spacing, typography, and image placement
- Practical use of web assets and external resources for styling and icons

## Why This Work Matters

This clone goes beyond copying a design. It is proof of the developer's ability to:

- Understand and interpret a finished UI design
- Recreate complex page structures without sacrificing performance
- Deliver an accurate and faithful implementation under challenge conditions
- Maintain readability and consistency across HTML and CSS
